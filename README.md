## 📊 Project Overview

This project presents an **interactive Power BI dashboard** designed to analyze **customer retention, churn behavior, and revenue trends for a SaaS business**.

The dashboard helps businesses understand customer behavior, identify churn risks, and track subscription revenue performance.

---

## 🎯 Key Objectives

* Analyze **customer churn trends**
* Track **monthly recurring revenue (MRR)**
* Identify **high-performing subscription plans**
* Understand **customer demographics**
* Visualize **geographic customer distribution**
